## 📌 스터디 규칙

### 운영 방식

- 온라인/오프라인 스위칭하면서 진행

<br/>

## 🔍 참여 방법

1.  `git status`  후 필요하다면  `git pull origin main`  명령어 실행
2.  각자의 폴더에 코드 작성(이름_문제.확장자)
3.  Remote repository에 push
4. 매주 수요일 각자의 코드 설명

<br/>

## 📁 Repository 폴더 구조

- 이름/이름_문제번호.확장자
### 💡 예시
`고승완_개인정보 수집 유효기간.py`
